# Contributing to the Glossary

Open a [new term](https://github.com/sane-av/sane-av-glossary/issues/new?template=new-term.yml) or [edit term](https://github.com/sane-av/sane-av-glossary/issues/new?template=edit-term.yml) issue. Maintainers will review and merge changes.
